# 🧹 System Cleaner

**System Cleaner** is a lightweight Windows utility designed to safely clean **TEMP** and **Prefetch** folders, helping free up disk space and keep your system tidy.

Built with **WPF (.NET)** and designed for **Windows 10 & 11**, it features a modern dark UI, tray support, and a simple one-click workflow.

---

## ✨ Features

- 🗑️ Clean **%TEMP%** directory safely  
- ⚡ Clean **Windows Prefetch** (administrator optional)  
- 🟢 Clear status feedback with visual indicators  
- 🔄 Restart as Administrator when required  
- 🔔 Tray icon support  
- 📦 Single-file installer  
- 🔍 Built-in update checker (GitHub-based)

## 🚀 Installation

1. Download the latest installer from **Releases**
2. Run `SystemCleanerSetup.exe`
3. Follow the on-screen instructions

> ⚠️ Cleaning **Prefetch** may require Administrator privileges.

---

## 🔄 Updates

System Cleaner checks for updates using a simple version file hosted on GitHub.

- Update source: `updates/version.txt`
- When a newer version is available, the app can open the latest release automatically.
